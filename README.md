# pyrnvapi
Python Client API which gets the results from the official RNV Start.Info API provided by Rhein-Neckar-Verkehr GmbH (RNV)

## Requirements
`python3`

## Installation
```
git clone https://github.com/MyRNVPlan/pyrnvapi.git
cd pyrnvapi
pip install .
```
